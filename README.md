# Emoji loader

Fake loading screen with... bouncing emojis 🤯 !!

Example: https://enric1994.github.io/emojiloader/

## How to use
1. Fork the repo!
2. Enable GitHub Pages on `Settings`
3. Edit `index.html`:

```
  var emojis = [' 😍 ', ' 📖 ', ' 🤔 ', ' 💡 ', ' x ', ' 🚀 ', ' 🤯 ', ' 📊 ', ' 🤓 '];
  var text = 'Loading...';
  var url = 'https://google.com';
  var time = '3';
  var background = '#ea93da';
  ```
